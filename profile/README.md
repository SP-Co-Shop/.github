
## 목표
대규모 데이터를 다루기 위한 목적을 위한 클론 프로젝트입니다.
데이터들은 크롤링을 통해 수집중에 있으며 실제 서비스가 동작하는 것처럼 Airflow와 Python을 이용하여 실제 사용자와 판매자가 행동하는 것처럼 구현할 것입니다.<br> 
또한 데이터들을 **안전하고 정확하게** 핸들링 하는 것과 **성능에 있어 최적화** 하는 것을 목표로 하는 것입니다.<br>
